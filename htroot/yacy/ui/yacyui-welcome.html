Overview

YaCy-UI is going to be a rich client for YaCy based on the existing YaCY-API. YaCy-UI makes heavy use of jQuery, JavaScript, Ajax, XML and XSLT. Currently YaCy-UI is at most alpha status, so I would be very greatful for feedback and bug reports!
